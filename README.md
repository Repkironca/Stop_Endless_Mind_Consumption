# 暈船計數器

## 安裝

> 現在只有給 Android 的版本

右邊（手機的應該在最下面）有一欄 [Releases](https://github.com/Repkironca/Stop_Endless_Mind_Consumption/releases)，在那邊選擇最新的一個版本點進去，然後你會在 Assets 的地方找到 .apk，自己載下來安裝，堆

## 簡介與使用說明！

### 簡介

每個暈船小狗都想像派大星一樣豁達。

<img width="258" height="195" alt="image" src="https://github.com/user-attachments/assets/eaf83e6f-672f-485c-8b68-9bb8014af947" />


但眾所皆知，無論你再怎麼等，小石頭就是不會動起來，更跑不贏小蝸。而有時候，你無法得知自己暈到的人，是不是一顆俗頭 🌑

我不知道你的狀況為何，或許你暈到一個千年神木看不懂空氣，你在白耗費心神；或許你暈到另一個暈船小狗，那當然她連看都不看你一眼；或許，你暈到一個不願意走出舒適圈的極端個體，想要跟她破冰，比在**嚴冬泳**渡穿越白令海峽還難

你不能永無止境地魔自己的情緒，一再反覆地破防！於是這個 app 誕生了，它很陽春，唯一的目標就是讓你的痛苦可視化。你可以先設定一個停損點 $L$ ( 預設是 50，我個人用 500 )：
> **從現在開始，只要這咖再讓你痛苦 $L$ 個單位時間，且情況沒有任何好轉，你就可以考慮下船了**（或考慮設一個更高的停損點，人性本賤，你是抖 M）

### 發起紀錄

單位時間預設 60 分鐘。每次你被暈船對象狠狠刺傷，就可以在主畫面的下面，點擊一個顏色發起紀錄，並留下一些文字記錄，如果你想

<img width="135" height="293" alt="image" src="https://github.com/user-attachments/assets/ec5a9aca-6b99-4095-9c83-7451e999b719" />

顏色分為藍紫紅三種，你可以自己給他們定義，或直接抄我的去用，當然前提是你要跟我差不多嚴重😀：
> - 藍色：你正在做其他事，但內心卡了根針。想法正在你腦子盤旋不去，時刻刺痛你的神經，你要有意識地強迫自己無視它，來繼續你本來的活動
> - 紫色：你暫時失去做事的能力，或你恰好在通勤之類沒事可做，腦子瘋狂尖叫，感覺自己在精神崩潰邊緣。杏仁核過度活躍，極端想法無法停止地產生，包含但不限於自殺、傷人、血祭。此時你的前額葉可能在和杏仁核打架，它一邊壓制極端想法，一邊譴責你的不正常與偏激
> - 紅色：你的痛苦轉移到身體上的不適，包含但不限於心悸、假性心絞痛、冒冷汗、頭痛、呼吸困難、過度換氣、喉嚨異物感、胃痙攣、胃食道逆流。在生理學上就是急性壓力潰堤的症狀啦
> 
> ( 但如果你是嚴重到失眠，或你的極端想法已經快變成行動了，那你最好是去掛一下身心科 )

兩次紀錄間最少要隔一個時間單位

### 設定

誒對就是最右上角的那個按鈕，你可以在那邊改變你的止損點、間隔時間，或是把你的紀錄清空之類的

### 歷史紀錄

右上角靠左的那個按鈕。點進去後會進入一個新頁面，你可以透過右上角來調整要使用月曆模式或時間軸模式來瀏覽，透過上下滑動調整時間。點擊任何一天，即可展開個紀錄的發起時間與當時留下的文字

- 月曆模式

類似 Github 的 Commmit History，每天的顏色會是你該日所有紀錄的嚴重程度平均，愈嚴重愈偏血紅

- 時間軸模式

每個橫條代表一天，著色位置代表你紀錄的相對時間
